<br>
Given an array containing n distinct numbers taken from 0, 1, 2, ..., n, find the one that is missing from the array.


#### Input
 - The input line will contain a set of integers, n (0 <= n <= 50), seperated by spaces

#### Output
 - A single integer, representing the missing number from the array.
<hr>

#### Sample 1
##### Input
```
3 0 1
```
##### Output
```
2
```

#### Sample 2
##### Input
```
9 6 4 2 3 5 7 0 1
```
##### Output
```
8
```
